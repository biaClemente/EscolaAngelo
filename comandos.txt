#Como CLONAR um repositório do GITHUB com o Visual Code
I No GITHUB abra o repositório e clique no botão CODE> Local copiar link do repositório
No Visual Code abra a pasta de seus projetos (exemplo: Documentos/Python/)
Abra o Terminal e digite o comando: git clone <cole aqui o link compiado
Tecle ENTER
No VS Code abra a pasta clonada
No terminal digite: git config user.name <digite aqui seu nome de usuário no Github>
 Tecle ENTER
Ainda no terminal digite: git config user.email <digite aqui seu email no GitHub>
 Tecle ENTER
